## Architecture decisions (settled in research; not to be re-litigated in EXECUTE)

- **R1 → inline rubric.** The FR5 version-decision rubric lives in `.planning/M4.5.E5-LAUNCH-KIT.md`, **not** a standalone `docs/versioning.md` (that path is the shelved E1 S4 artifact — creating it would ship a stale stub or invite scope-creep; mirrors the D-E3-2 `compatibility.md` removal). Rubric is labeled "Signal's adopted convention" (semver.org is silent on the 0.x minor/patch split).
- **R2 → three channels.** GitHub Release (durable anchor, links the post) + direct 1:1 peer share (tester brief + Mac caveat) + README surfacing at publish time. HN/Reddit/X explicitly held back (named in the handoff as a conscious deferral).
- **Doc homes.** Public/peer-facing → `docs/` (`launch-post.md`, `demo-script.md`, `tester-brief.md`). Internal launch-ops → `.planning/M4.5.E5-LAUNCH-KIT.md` (rubric + release-notes draft + handoff checklist + channels).
- **README link to the post is a publish-time handoff action, NOT an EXECUTE task** — the post is a publish-when-ready draft (D-E5-3); linking a draft into the stranger-facing README is premature. (E5's ACs carry no README-link requirement, unlike E4's AC6.)
- **Docs-Epic Nyquist posture.** "Tests" = automated link-integrity + word-count + structural-presence checks over the new docs, plus the validator subprocess smoke test — **not** code TDD. One growing guard file: `tests/e5-launch-assets.test.js`. Each doc-slice adds its assertions and the doc that satisfies them in the same slice, so the root suite stays green at every checkpoint.
- **Bonus accuracy fix.** When E5 touches the CHANGELOG (S3.t8), soften the dangling `CHANGELOG.md:3` reference to `docs/versioning.md` (a shelved, non-existent artifact) → point at `SECURITY.md`'s pre-1.0 line + the inline rubric.
- **Accuracy guardrails for prose (carry into every doc):** lead the post with **7 analyzed plugins**; v1 **ports two** (GSD + Agent Skills); v2 items are roadmap, not shipped; no invented users/testimonials/benchmarks; privacy line copied verbatim-in-spirit from `README.md:181–185`; Mac-only caveat from `README.md:32`.
